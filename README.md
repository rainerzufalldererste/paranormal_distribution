# paranormal_distribution
### A tiny, fast, header-only weighted integer distribution for random value sequences.

- Creates a random `int8_t` values weighted around 0 from 16-22 random bits
- 7 different smoothing levels (with identical performance characteristics)
- Written in C++
- Branchless, SIMD friendly Algorithm
